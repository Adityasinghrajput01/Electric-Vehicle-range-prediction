- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Machine Learning Model Training
- Model Evaluation
- Business Insights

- Live Power BI Report

# 📈 Business Insights

- Battery capacity is the strongest factor influencing driving range.

- Premium EVs generally offer significantly higher range.

- Fast charging vehicles attract better consumer preference.

- Larger battery packs improve range but increase vehicle cost.

- Vehicle efficiency plays a major role in maximizing driving distance.

---

# 🌿 Environmental Insights

- Higher EV adoption can reduce carbon emissions.

- Improved battery technology contributes to sustainable transportation.

- Range prediction helps consumers choose the most suitable EV.

---

# 📊 Model Evaluation

The project compares multiple Machine Learning models using evaluation metrics such as:

- MAE
- RMSE
- R² Score

The best-performing model is selected for final predictions.
